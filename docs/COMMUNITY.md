# Komunitas Gorontalo Unite

Gorontalo Unite adalah proyek **open source berbasis komunitas** yang dibangun dari, oleh, dan untuk masyarakat Gorontalo. Tidak perlu bisa coding untuk berkontribusi!

## Apa yang Bisa Kamu Kontribusikan

### 📝 Tanpa Coding
- [Fakta & pengetahuan tentang Gorontalo](./CONTRIBUTE_RAG.md) — tempat wisata, kuliner, budaya, sejarah, dll
- [Kosakata & dialek bahasa Gorontalo](./CONTRIBUTE_DIALECT.md) — kamus, percakapan, peribahasa, dll
- Laporan bug — "ini tidak berfungsi di hp saya"
- Usulan fitur — "enak kalau ada fitur X"
- Testing — coba fitur baru dan beri feedback
- Review artikel — cek akurasi konten yang dipublish

### 💻 Dengan Coding
- Implementasi fitur dari [backlog](../README.md#roadmap--backlog)
- Bug fix
- Optimasi performa
- Peningkatan aksesibilitas
- Penulisan test

### 🎨 Desain
- Mockup UI untuk fitur baru
- Aset visual (ikon, ilustrasi)
- Perbaikan UX yang terasa aneh

## Program Apresiasi Kontributor

### 🏆 Tingkatan Kontributor

| Tingkat | Syarat | Keuntungan |
|---|---|---|
| **Kontributor** | 1 kontribusi diterima | Nama di CONTRIBUTORS.md |
| **Kontributor Aktif** | 10+ kontribusi | Nama di halaman /about website |
| **Mitra Komunitas** | 50+ kontribusi atau kualitas luar biasa | Badge khusus |
| **Core Contributor** | Kontribusi konsisten + trusted | Akses langsung ke main, diajak diskusi roadmap |

### 📊 Apa yang Dihitung sebagai "Kontribusi"?
- 1 Pull Request yang di-merge = 1+ kontribusi (tergantung ukuran)
- 1 data RAG yang diterima = 1 kontribusi
- 10 kosakata = 1 kontribusi
- 1 laporan bug yang berhasil diperbaiki = 1 kontribusi
- Review PR yang substantif = 0.5 kontribusi

## Pertanyaan yang Sering Ditanya

**Q: Saya bukan programmer, apakah bisa berkontribusi?**
A: Sangat bisa! Kontribusi data (fakta lokal, bahasa Gorontalo) sama berharganya dengan kontribusi kode. AI kami tidak akan cerdas tanpa data dari orang Gorontalo asli.

**Q: Saya ingin berkontribusi tapi tidak punya GitHub. Bagaimana?**
A: Segera tersedia: form Google Form untuk kontribusi data tanpa GitHub. Pantau di halaman website atau follow media sosial kami.

**Q: Apakah website ini gratis?**
A: Ya, sepenuhnya gratis untuk pengguna. Proyek ini open source dan dikelola secara sukarela.

**Q: Saya punya informasi rahasia/sensitif tentang Gorontalo. Boleh dikontribusikan?**
A: Jangan. Hanya kontribusikan informasi yang sudah bersifat publik dan bisa diverifikasi. Jangan sertakan informasi pribadi orang lain tanpa izin.

**Q: Bagaimana jika ada yang mencuri ide/kontribusi saya?**
A: Semua kontribusi tercatat di GitHub dengan timestamp dan nama. Lisensi proyek (MIT) melindungi hak kontributor. Nama kontributor akan selalu dicantumkan.
